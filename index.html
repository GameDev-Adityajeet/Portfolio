Shadow Realm
UnityC#Shader Graph3D RPG

Project Overview

Shadow Realm is a fully-featured 3D action RPG built in Unity. The game features a procedural dungeon system, a responsive combat system with combo chains, and a branching storyline with multiple endings.

Key Features

  • Procedural dungeon generation using Wave Function Collapse algorithm
  • Custom inverse kinematics for character animation
  • Fully optimized LOD system supporting 200+ NPCs simultaneously
  • Real-time day/night cycle with volumetric lighting

Null Signal
Unreal Engine 5C++LumenFPS

Project Overview

Null Signal is an atmospheric sci-fi horror FPS built in Unreal Engine 5. It leverages Lumen global illumination and Nanite geometry to create a photorealistic and terrifying experience.

Key Features

  • Dynamic AI using Behaviour Trees and hierarchical state machines
  • Full Unreal 5 Lumen + Nanite pipeline
  • Volumetric fog and dynamic audio mix
  • Custom Chaos physics destruction system

Hex Tactics
Three.jsWebGLJavaScriptStrategy

Project Overview

Hex Tactics is a real-time browser-based strategy game. Built entirely with Three.js and WebGL for buttery-smooth 60fps rendering without any native plugin.

Key Features

  • Procedural hex grid map generation
  • Custom WebGL shader for hex cell selection and hover glow
  • Socket-based real-time multiplayer (up to 4 players)
  • Works on both desktop and mobile browsers
