Overview

Legend of Indra vs Vritrasur is a long-form action-adventure RPG built in Unreal Engine 5, inspired by the ancient Hindu epic of the war between Indra, the king of gods, and Vritrasur, the cosmic dragon demon.

I served as both Game Designer and Developer on this project — responsible for the combat system, enemy AI, narrative flow, and environment design. The game was built and iterated across multiple months and showcased at the CU Project Expo 2025, where it won both 1st and 3rd Prize.

The project features 3D immersive support with atmospheric Unreal Engine 5 lighting, Blueprint-driven gameplay systems, and a deeply balanced progression curve crafted through extensive iterative playtesting.

Key Features

  • Custom combo-based combat system with ability cooldowns and hit-stop feedback for satisfying melee encounters
  • Enemy AI designed with patrol, alert, and attack behaviour states using Unreal Engine 5 Behaviour Trees
  • Full narrative arc designed via comprehensive GDD — including branching dialogue and contextual cinematics
  • Combat balancing and gameplay progression tuned through multiple rounds of iterative playtesting
  • 3D immersive environments with real-time atmospheric lighting powered by Unreal Engine 5's Lumen system
  • Blueprint Visual Scripting used throughout for rapid iteration and clean modular logic
