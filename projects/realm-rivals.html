Overview

Realm Rivals is a small but focused multiplayer arena game built in Unreal Engine 5 using Blueprints. The game operates on a LAN network — players connect under the same Wi-Fi connection to enter an arena where they can equip gear and engage in competitive combat.

I was responsible for creating the damage system — handling hit detection, damage calculation, and player feedback — as well as the interaction system that allows players to equip weapons and gear before entering the arena. These systems work together to create a pick-up-and-play competitive experience.

In addition to the core gameplay systems, I designed and built several levels for the arena, each offering different strategic layouts and combat opportunities for the players.
