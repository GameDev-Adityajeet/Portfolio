Overview

Reality Run was developed in 48 hours for Global Game Jam 2026. The concept blends the tension of an escape room with the action of a third-person shooter — players must navigate an environment under constant threat, using stealth and combat to survive and escape.

The standout feature is the Alertness Level Detector — a dynamic system that tracks how aware guards and enemies are of the player's presence. As alertness rises, the world responds: guards become more aggressive, patrols shift, and escape routes close off.

The game was awarded the Best Gameplay Factor award at Global Game Jam 2026, praised by judges for its innovative alert mechanic, smooth interaction system, and polished feel given the 48-hour constraint.

Key Features

  • Alertness Level Detector — a dynamic guard awareness system that scales difficulty based on player visibility and actions
  • Full interaction system allowing the player to use objects, doors, switches, and NPCs in a context-sensitive way
  • Dynamic menus with in-world UI elements that respond to game state and player alertness levels
  • Third-person shooter mechanics with cover system and stealth movement options for varied play styles
  • Escape-room puzzle design integrated seamlessly with action gameplay for a unique hybrid experience
  • Awarded Best Gameplay Factor at Global Game Jam 2026 — delivered as a complete, polished title in 48 hours
