🥉 Global Game Jam 2025 — 3rd Prize
Unity · 2D Arcade · Built in 48 Hours

BubbleBoom

A fast-paced 2D bubble-shooting arcade game built entirely in 48 hours for Global Game Jam 2025. Chain reactions, escalating difficulty, and addictive gameplay loops — all shipped from zero in under two days.

Unity C# 2D Physics Game Jam Particle System Procedural Difficulty
Play on itch.io GitHub

Overview

BubbleBoom was created for Global Game Jam 2025 within a strict 48-hour development sprint. The theme drove the design toward a bubble-shooting mechanic where groups of same-coloured bubbles burst together, creating chain-reaction combos.

As the sole designer and developer, I had to rapidly scope, design, build, and polish the game within the jam window. The result was a tightly designed arcade experience with clear progression, snappy controls, and satisfying visual feedback on every shot.

The project was awarded 3rd Prize at Global Game Jam 2025, recognised for its polished feel and engaging gameplay loop considering the tight time constraint.
